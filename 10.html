Let me say one more thing about measures.
How would we write \(\displaystyle\int_{x=0}^{x=4}f(x)\,\text{d}u\), if we could only use the variable \(x\)? Well, just as we know that \(\text{d}x=2u\,\text{d}u\), we can also write \(\text{d}u=\frac{1}{2u}\text{d}x=\frac{1}{2\sqrt{x}}\text{d}x\). I'm sure you've seen something like this in your high school math classes. So, we'd write \(\displaystyle\int_{x=0}^{x=4}\frac{1}{2\sqrt{x}}f(x)\,\text{d}x\).
This is interesting. Measuring the area of this continuous function in a weird way defined by \(\text{d}u\) is exactly the same as multiplying the original function by \(\frac{1}{2\sqrt{x}}\) and then doing the normal area calculation.
We can in fact say this of measures in general. There's no time explain in full, but I'll just say that integrating a continuous function \(f(x)\) over a "usable" measure on \([0,4]\) (in technical terms, a measure which is absolutely continuous with respect to the Lebesgue measure) is the same as multiplying \(f(x)\) by a "usable" function \(D(x)\) (a Lebesgue-integrable function) and then integrating \(f(x)D(x)\) in the normal way.
